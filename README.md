## Описание

Проект "Виджеты для банковских операций" представляет собой софт для быстрой и удобной работы с банковскими операциями.
Он маскирует счета, сортирует операции по датам и статусу. 

Находится в разработке.
## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/mitraslav/bank_operations_widgets
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
