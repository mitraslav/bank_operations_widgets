## Описание

Проект "Виджеты для банковских операций" представляет собой софт для быстрой и удобной работы с банковскими операциями.
Он маскирует счета, сортирует операции по датам и статусу. 

Находится в разработке.
## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/mitraslav/bank_operations_widgets
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование

1. Модуль **masks** содержит функции **get_mask_card_number** и **get_mask_account**, которые принимают номер карты и 
номер счета соответственно и возвращают их маски.


2. Модуль **processing** содержит функции **filter_by_state** и **sort_by_date**,
созданные для фильтрации списка операций по заданному пользователем состоянию и 
сортировки списка операций по дате соответственно.


3. Модуль **widget** содержит функции **mask_account_card** и **get_date**, созданные для маскировки номеров карт и счетов 
пользователей и возвращения строки с датой в формате ДД.ММ.ГГГГ соответственно.


4. Модуль **generators** содержит функции **filter_by_currency**, **transaction_descriptions** и **card_number_generator**,
созданные для фильтрации транзакций по заданной валюте и попеременном выводе их в консоль, попеременном возвращении
поданных на вход банковских операций и генерации номеров карт в заданном диапазоне соответственно.
## Тестирование

Для проверки корректности работы проекта реализованы модульные тесты с использованием `pytest`.

### Как запустить тесты
```
pytest tests
```
До этого нужно загрузить фреймворк pytest через poetry:

```
poetry add --group dev pytest
```
